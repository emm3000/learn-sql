# Dudas y Aclaraciones: INSERT

## Duda 1

**Tengo que poner todas las columnas siempre**

No siempre. Puedes omitir columnas si tienen `DEFAULT`, permiten `NULL` o si la tabla lo admite.

## Duda 2

**Por que usar `RETURNING`**

Sirve para ver inmediatamente que se guardo, sin hacer otra consulta aparte.

## Duda 3

**Puedo insertar varias filas a la vez**

Si. Es una forma comun y eficiente de cargar datos.
