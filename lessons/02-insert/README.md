# Leccion 02: INSERT

## Objetivo

Aprender a guardar datos nuevos en PostgreSQL con seguridad y claridad.

## Lo que debes dominar

- insertar una fila
- insertar varias filas
- entender el orden de columnas y valores
- usar `RETURNING`
- evitar errores comunes con tipos y `NULL`

## Flujo de estudio

1. Leer la teoria breve.
2. Correr los ejemplos en `psql`.
3. Resolver los ejercicios.
4. Revisar los casos reales.
5. Escribir tus dudas y aclaraciones.
