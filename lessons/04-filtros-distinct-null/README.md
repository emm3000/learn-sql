# Leccion 04: Filtros, DISTINCT y NULL

## Objetivo

Aprender a filtrar datos de forma precisa y entender como se comportan los valores nulos y los duplicados.

## Lo que debes dominar

- `WHERE`
- operadores de comparacion
- `AND`
- `OR`
- `NOT`
- `IN`
- `BETWEEN`
- `LIKE`
- `IS NULL`
- `IS NOT NULL`
- `DISTINCT`

## Flujo de estudio

1. Leer la teoria breve.
2. Correr los ejemplos en `psql`.
3. Resolver los ejercicios.
4. Revisar los casos reales.
5. Escribir tus dudas y aclaraciones.
