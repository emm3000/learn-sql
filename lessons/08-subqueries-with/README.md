# Leccion 08: Subqueries y WITH

## Objetivo

Aprender a resolver preguntas complejas usando consultas dentro de consultas y CTEs para hacer el SQL mas legible.

## Lo que debes dominar

- subqueries en `SELECT`
- subqueries en `WHERE`
- subqueries en `FROM`
- `WITH` / CTEs
- dividir una consulta grande en pasos
- reutilizar resultados intermedios

## Flujo de estudio

1. Cargar el seed clasico.
2. Leer la teoria breve.
3. Correr los ejemplos en `psql`.
4. Resolver los ejercicios.
5. Revisar los casos reales.
6. Escribir tus dudas y aclaraciones.

## Seed recomendado

```bash
docker compose exec -T db psql -U postgres -d aprendizaje < seed/classic_company.sql
```
