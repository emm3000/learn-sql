# Leccion 06: JOIN

## Objetivo

Aprender a combinar datos de varias tablas para resolver preguntas reales sobre empleados, departamentos, clientes, pedidos y proyectos.

## Lo que debes dominar

- `INNER JOIN`
- `LEFT JOIN`
- relacion entre tablas
- claves foraneas
- alias de tablas
- unir por columnas correctas

## Flujo de estudio

1. Cargar el seed clasico.
2. Leer la teoria breve.
3. Correr los ejemplos en `psql`.
4. Resolver los ejercicios.
5. Revisar los casos reales.
6. Escribir tus dudas y aclaraciones.

## Seed recomendado

```bash
docker compose exec -T db psql -U postgres -d aprendizaje < seed/classic_company.sql
```
