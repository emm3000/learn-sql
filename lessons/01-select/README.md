# Leccion 01: SELECT

## Objetivo

Aprender a leer datos de PostgreSQL con consultas simples y claras.

## Lo que debes dominar

- seleccionar columnas
- filtrar filas
- ordenar resultados
- limitar resultados
- entender `NULL`

## Flujo de estudio

1. Leer la teoria breve.
2. Correr los ejemplos en `psql`.
3. Resolver los ejercicios.
4. Revisar los casos reales.
5. Escribir tus dudas y aclaraciones.
