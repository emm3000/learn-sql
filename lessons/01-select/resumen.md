# Resumen: SELECT

## Lo esencial

- `SELECT` lee datos.
- `WHERE` filtra filas.
- `ORDER BY` ordena.
- `LIMIT` recorta resultados.

## Lo que debe salirte natural

- leer una tabla completa
- leer solo algunas columnas
- buscar registros concretos
- revisar resultados recientes

## Siguiente paso

Cuando esto ya salga bien, pasamos a:
- `INSERT`
- `UPDATE`
- `DELETE`
