# Leccion 05: ORDER BY y LIMIT

## Objetivo

Aprender a ordenar resultados y limitar cuantas filas devuelve una consulta, algo basico para explorar datos y construir pantallas o reportes.

## Lo que debes dominar

- `ORDER BY`
- orden ascendente y descendente
- ordenar por una o varias columnas
- `LIMIT`
- `OFFSET`
- paginacion basica

## Flujo de estudio

1. Leer la teoria breve.
2. Correr los ejemplos en `psql`.
3. Resolver los ejercicios.
4. Revisar los casos reales.
5. Escribir tus dudas y aclaraciones.
