# Resumen: ORDER BY y LIMIT

## Lo esencial

- `ORDER BY` ordena resultados.
- `LIMIT` limita cuantas filas salen.
- `OFFSET` desplaza el punto de inicio.
- ordenar antes de limitar da resultados mas predecibles.

## Lo que debe salirte natural

- ver registros recientes
- ordenar para explorar datos
- paginar resultados
- usar varias columnas para desempatar

## Siguiente paso

Cuando esto ya salga bien, pasamos a:

- `JOIN`
- relaciones entre tablas
