# Leccion 03: UPDATE y DELETE

## Objetivo

Aprender a modificar y borrar datos con cuidado, usando filtros correctos para no tocar mas de lo necesario.

## Lo que debes dominar

- actualizar una fila
- actualizar varias filas
- borrar una fila
- borrar varias filas
- usar `WHERE` con cuidado
- usar `RETURNING`
- evitar cambios accidentales

## Flujo de estudio

1. Leer la teoria breve.
2. Correr los ejemplos en `psql`.
3. Resolver los ejercicios.
4. Revisar los casos reales.
5. Escribir tus dudas y aclaraciones.
